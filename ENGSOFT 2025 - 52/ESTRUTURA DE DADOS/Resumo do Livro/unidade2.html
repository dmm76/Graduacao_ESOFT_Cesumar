Unidade 2 – Pilhas e Filas

Pilhas

A pilha é uma estrutura de dados do tipo LIFO (Last In, First Out).

Exemplo real: Empilhar livros ou caixas – o último que entra é o primeiro que sai.

#define TAM 10
struct Pilha {
  int dados[TAM];
  int topo;
};

Operações:

  • push: inserir no topo
  • pop: remover do topo

Filas

A fila é uma estrutura de dados do tipo FIFO (First In, First Out).

Exemplo real: Fila de banco – o primeiro que entra é o primeiro a ser atendido.

#define TAM 10
struct Fila {
  int dados[TAM];
  int ini, fim;
};

Operações:

  • enqueue: inserir no final
  • dequeue: remover do início
