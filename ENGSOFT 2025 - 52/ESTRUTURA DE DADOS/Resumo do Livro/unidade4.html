Unidade 4 – Grafos

Conceito

Um grafo é uma estrutura composta por vértices (nós) e arestas (ligações entre nós).

Representações:

  • Matriz de adjacência: usa matriz para indicar ligações
  • Lista de adjacência: lista ligada para cada vértice com seus vizinhos

Exemplo:

int grafo[5][5]; // grafo com 5 vértices
