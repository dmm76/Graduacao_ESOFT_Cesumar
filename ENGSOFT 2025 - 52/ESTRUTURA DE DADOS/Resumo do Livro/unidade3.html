Unidade 3 – Listas Dinâmicas

Listas Encadeadas

Permitem alocação dinâmica e inserção/remoção eficiente sem realocação em memória.

struct No {
  int valor;
  struct No* prox;
};

Variações:

  • Lista simples: apenas um ponteiro para o próximo
  • Lista duplamente encadeada: ponteiros para anterior e próximo
  • Lista circular: o último aponta para o primeiro
