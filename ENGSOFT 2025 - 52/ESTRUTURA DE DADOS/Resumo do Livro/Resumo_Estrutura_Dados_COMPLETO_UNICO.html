Unidade 2 – Pilhas e Filas

Resumo – Unidade 2: Pilhas e Filas

1. Conceitos Iniciais
Pilhas (LIFO) e Filas (FIFO) são estruturas que controlam a ordem de entrada e saída de dados.

2. Pilhas
Usa vetor e um topo.
Operações:
- push: insere no topo
- pop: remove do topo

Código:
#define TAM 10
struct Pilha { int dados[TAM]; int topo; };

3. Filas
Usa vetor com início e fim.
Operações:
- enqueue: insere no fim
- dequeue: remove do início

Código:
#define TAM 10
struct Fila { int dados[TAM]; int ini, fim; };

4. Comparação:
| Estrutura | Inserção | Remoção |
|-----------|----------|---------|
| Pilha     | Topo     | Topo    |
| Fila      | Final    | Início  |

Unidade 3 – Listas Dinâmicas

Resumo – Unidade 3: Listas Dinâmicas

1. Introdução
Listas ligadas usam nós encadeados por ponteiros.

2. Lista Simples
Cada nó aponta para o próximo.
struct No { int valor; struct No* prox; };

3. Lista Dupla
Aponta para anterior e próximo.
struct No { int valor; struct No* ant, *prox; };

4. Lista Circular
Último nó aponta para o primeiro.

5. Comparativo:
| Tipo de Lista           | Ponteiros          | Navegação |
|-------------------------|--------------------|-----------|
| Simples Encadeada       | Próximo            | Frente    |
| Duplamente Encadeada    | Anterior e Próximo | Dois lados|
| Circular Simples        | Próximo            | Circular  |

Unidade 4 – Grafos

Resumo – Unidade 4: Grafos

1. Conceito
Grafo = vértices + arestas

2. Representações:
a) Matriz de Adjacência:
int grafo[5][5];
grafo[i][j] = 1 indica conexão.

b) Lista de Adjacência:
struct No { int vertice; struct No* prox; };

3. Tipos:
- Dirigido / Não-dirigido
- Ponderado / Não-ponderado

4. Aplicações: redes, mapas, redes sociais.

Unidade 5 – Busca em Grafos

Resumo – Unidade 5: Busca em Grafos

1. DFS – Busca em Profundidade
Visita um caminho até o fim antes de voltar.

2. BFS – Busca em Largura
Visita todos os vizinhos antes de avançar.

3. Algoritmo de Dijkstra
Menor caminho entre dois vértices com pesos positivos.

Usa vetor de distâncias:
dist[v] = menor caminho de origem até v
