Resumo - Estrutura de Dados I (Unicesumar)

Unidade 1: Ponteiros

1. Estruturas Homogêneas e Heterogêneas

  • Homogêneas: armazenam um único tipo de dado. Exemplo: int numeros[5];
  • Heterogêneas: armazenam vários tipos. Exemplo: struct Pessoa {
      char nome[50];
      int idade;
    };

2. Vetores e Matrizes

int dados[5]; float matriz[2][3];

3. Registros (Structs)

struct Fraction {
  int numerator;
  int denominator;
  float value;
};

4. Ponteiros

int x = 10;
int *ptr = &x;
printf("%d", *ptr); // imprime 10

5. Propriedades dos Ponteiros

  • Permitem navegação de memória: ptr + 1 aponta 4 bytes à frente (int).

6. Alocação Dinâmica

int *ptr = (int *) malloc(sizeof(int));
*ptr = 42;

7. Vetores Dinâmicos

int tam;
scanf("%d", &tam);
int *vetor = (int *) malloc(tam * sizeof(int));
for (int i = 0; i < tam; i++) vetor[i] = pow(2, i);

Questões Frequentes

  • 1. Qual a diferença entre ponteiro e variável comum?
    Variável guarda valor. Ponteiro guarda endereço.
  • 2. O que acontece ao usar *ptr sem alocação?
    Erro de execução (acesso a endereço inválido).
  • 3. Quando usar alocação dinâmica?
    Quando o tamanho dos dados é incerto durante a codificação.
